# backend-webhook

Решение тестового задания **webhook платежей** и **активацию/продление подписки** (PostgreSQL + Node.js псевдокод).

## Что открыть в первую очередь

- **`solution.md`** — основной сдаваемый документ: схема таблиц/уникальности/индексы, псевдокод обработчика, edge cases, логи и метрики.

## Остальные файлы (по необходимости)
- **`edge-cases.md`** — разбор edge cases из задания и что делать в каждом (в т.ч. падение сервера “посередине”).
- **`observability.md`** — минимальные логи/метрики/алерты и подход к дебагу через сохраненный payload.
